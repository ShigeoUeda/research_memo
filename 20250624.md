## **【仮説】企画のずらし**

企画は、既存のジャンルが持つ「暗黙の構成要素のパッケージ」を分解し、**部品を意図的に入れ替える（ずらす）** ことで生まれることが多い。
そして、この**ずらし**は、**方向性**と**変化の大きさ**の2つの軸で分析できます。

### 「ずらし」の具体的パターン

| ずらしパターン | 説明 | 具体例 |
|---|---|---|
| **対象の転換** | 番組が焦点を当てる主役を変える | グルメ番組で「料理」から「店主の人間性」へ |
| **役割の反転** | 通常の役割分担を逆転させる | 「タレントが質問、専門家が回答」→「専門家が質問、タレントが体験」 |
| **要素の移植** | 他ジャンルの要素を持ち込む | 旅番組に「競争ルール」を導入（ローカル路線バス） |
| **演出の削減/増強** | 既存の演出手法を極端に減らす/増やす | ナレーション過多→ミニマル演出 |
| **時間軸の変更** | 通常の時間の流れを変える | リアルタイム→長期密着、瞬間→継続観察 |

### 変化の大きさ判定基準

以下3つの指標で総合評価し、客観性を高めます：

#### A. 視聴者の期待値からのずらし
- ★☆☆：そのジャンルを見る視聴者が「想定内」と感じる範囲
- ★★☆：「こんな展開があるのか」と驚くが理解できる範囲  
- ★★★：「これは同じジャンルなのか？」と困惑するレベル

#### B. 業界内での前例からのずらし
- ★☆☆：類似の試みが過去に複数存在
- ★★☆：部分的に前例があるが、組み合わせ方が新しい
- ★★★：業界初、または極めて稀な試み

#### C. 制作手法・リソースからのずらし
- ★☆☆：既存の制作体制・予算配分で対応可能
- ★★☆：一部の制作プロセスや人員配置を大幅変更
- ★★★：従来とは全く異なる制作体制・技術が必要

| 軸 | 説明 |
| :---- | :---- |
| **方向性** | 「どのクラスの、どの要素を、何へ」ずらしたかを示す変化のベクトル。</br>・【視聴感情】のずらし: (例: 「体験」中心から「共感」中心へ)</br>・【フォーマット】のずらし: (例: 「対決型」のルールを一部改変する)</br>・【構成要素】のずらし: (例: 「タレント」の役割を「スタッフ」へずらす) |
| **変化の大きさ** | 上記A・B・Cの3指標で総合評価。主観性を抑えつつ、企画の斬新さを測る指標となります。 |

**「ずらし」は企画を構成する個別の要素（視聴感情・フォーマット・構成要素の各項目）に対して作用する**
この考え方をもとに前回の分析の結果を再考してみた。
