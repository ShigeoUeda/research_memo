## **【仮説】企画のずらし**

企画は、既存のジャンルが持つ「暗黙の構成要素のパッケージ」を分解し、**部品を意図的に入れ替える（ずらす）** ことで生まれることが多い。
そして、この**ずらし**は、**方向性**と**変化の大きさ**の2つの軸で分析できます。

| 軸 | 説明 |
| :---- | :---- |
| **方向性** | 「どのクラスの、どの要素を、何へ」ずらしたかを示す変化のベクトル。</br>・【視聴感情】のずらし: (例: 「体験」中心から「共感」中心へ)</br>・【フォーマット】のずらし: (例: 「対決型」のルールを一部改変する)</br>・【構成要素】のずらし: (例: 「タレント」の役割を「スタッフ」へずらす) |
| **変化の大きさ** | 元のパターンからどのくらい離れているかを示す変化の度合い。主観性を伴いますが、3段階で評価することで企画の斬新さを測る指標となります。</br>・★☆☆ (微調整): 既存フォーマットの範囲内でのマイナーチェンジ。</br>・★★☆ (再解釈): ジャンルの常識や前提を覆し、「こう来たか」という驚きを生む大胆な変更。</br>・★★★ (パラダイムシフト): もはや元のジャンルとは別物と言えるほど革新的で、新たな視聴体験やジャンルそのものを創出する変更。 |

**「ずらし」は企画を構成する個別の要素（視聴感情・フォーマット・構成要素の各項目）に対して作用する**
この考え方をもとに前回の分析の結果を再考してみた。
