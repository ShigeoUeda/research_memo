# TRIZ 40の発明原理

| No. | 原理名 | 説明 | カスタム採用GPT件数 |
|-----|--------|------|----|
| 1 | 分割 | 対象を独立した部分に分ける、または分解可能にする | 　　　|
| 2 | 分離 | 対象から邪魔な部分や必要な部分を取り除く | 　　　|
| 3 | 局所的性質 | 対象の各部分を最適な条件で機能させる | 　　　|
| 4 | 非対称 | 対象の対称形を非対称形に変える | 　　　|
| 5 | 統合 | 同質または類似の対象や操作を空間的・時間的に統合する | 　　　|
| 6 | 汎用性 | 対象に複数の異なる機能を持たせ、他の部分を不要にする | 　　　|
| 7 | 入れ子 | ある対象を別の対象の中に入れ、さらにその対象を別の対象の中に入れる | 　　　|
| 8 | 釣り合い | 対象に作用する力を打ち消す別の力を加える | 　　　|
| 9 | 先取り反作用 | 有害な作用が予想される場合、事前に反対作用を加える | 　　　|
| 10 | 先取り作用 | 必要な作用を事前に全体的または部分的に実行する | 　　　|
| 11 | 事前保護 | 対象の信頼性が低い場合、事前に緊急手段を準備する | 　　　|
| 12 | 等ポテンシャル | 対象を持ち上げたり下げたりする必要がない条件で作業する | 　　　|
| 13 | 逆発想 | 問題を解決する通常の作用の代わりに反対の作用を使う | 　　　|
| 14 | 曲面 | 直線部分を曲線に、平面を球面に変える | 　　　|
| 15 | ダイナミック性 | 対象や環境の特性を各段階で最適になるように変化させる | 　　　|
| 16 | アバウト | 完全な解決が難しい場合、少し過剰または不足した解決を行う | 　　　|
| 17 | 他次元移行 | 1次元の移動を2次元、2次元を3次元に変える | 　　　|
| 18 | 機械的振動 | 対象を振動させる、振動数を増加させる | 　　　|
| 19 | 周期的作用 | 連続的作用を周期的または脈動的作用に変える | 　　　|
| 20 | 有効作用の継続 | 作用を中断なく継続する、対象の全ての部分を常にフル稼働させる | 　　　|
| 21 | 高速実行 | 有害または危険な操作を高速で実行する | 　　　|
| 22 | 災いを転じて福となす | 有害な要因や作用を利用して有益な効果を得る | 　　　|
| 23 | フィードバック | フィードバックを導入する、すでにある場合は変更する | 　　　|
| 24 | 仲介 | 仲介する対象を利用する、または一時的に別の対象を連結する | 　　　|
| 25 | セルフサービス | 対象が自分自身を補助したり修復したりできるようにする | 　　　|
| 26 | 代替 | 高価で複雑な対象を安価でシンプルな対象で代替する | 　　　|
| 27 | 使い捨て | 安価な対象で高価な対象を代替し、特定の性質を犠牲にする | 　　　|
| 28 | 機械的システムの代替 | 機械的システムを視覚、聴覚、嗅覚などのシステムに代替する | 　　　|
| 29 | 空気圧・水圧構造 | 固い部分を気体や液体に代替する | 　　　|
| 30 | 柔軟な殻・薄膜 | 通常の構造の代わりに柔軟な殻や薄膜を使う | 　　　|
| 31 | 多孔質材料 | 対象を多孔質にする、または多孔質の追加要素を使う | 　　　|
| 32 | 色の変化 | 対象や環境の色を変える、対象の透明度を変える | 　　　|
| 33 | 均質性 | 対象と相互作用する対象を同じ材料または類似の性質を持つ材料にする | 　　　|
| 34 | 排除と再生 | 機能を果たした部分を廃棄・溶解・蒸発させる、または使用中に再生する | 　　　|
| 35 | パラメータの変更 | 対象の物理的状態を変える(温度、体積、柔軟性など) | 　　　|
| 36 | 相変化 | 相転移時に生じる現象を利用する(体積変化、熱の吸収・放出など) | 　　　|
| 37 | 熱膨張 | 材料の熱膨張または熱収縮を利用する | 　　　|
| 38 | 強い酸化剤 | 通常の空気を富化空気に、富化空気を純粋な酸素に代替する | 　　　|
| 39 | 不活性雰囲気 | 通常の環境を不活性環境に代替する、真空を加える | 　　　|
| 40 | 複合材料 | 均質な材料を複合材料に代替する | 　　　|
